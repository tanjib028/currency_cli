A fast currency converter built in Rust using the apilayer Exchange Rates Data API.

Features

  • Convert between any two currencies
  • Historical conversions by date
  • Multi-currency conversions
  • Output formats: plain, JSON, CSV

Usage Examples

currency-cli convert 100 USD EUR
currency-cli convert 120.5 AUD CAD --date 2020-05-12
currency-cli multi 50 GBP USD EUR JPY
